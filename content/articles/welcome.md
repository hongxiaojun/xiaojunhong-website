## Technical Stack

This website is built with:
- **Hugo** - A fast static site generator
- **Cloudflare Pages** - For hosting and deployment
- **GitHub** - For version control and CI/CD

As a beginner to web deployment, I chose this stack because:
1. Hugo is simple and well-documented
2. Cloudflare Pages offers free hosting with SSL
3. Everything can be managed with git commands
