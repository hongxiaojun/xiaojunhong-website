## 🚀 立即开始部署

### 第一步：安装 Hugo（必需）

```bash
brew install hugo
```

### 第二步：本地测试

```bash
cd /Users/add/xiaojunhong-website
./setup.sh
```

然后在浏览器打开：http://localhost:1313

### 第三步：推送到 GitHub

**选项 A：使用自动化脚本（推荐）**

```bash
./github-setup.sh
```

**选项 B：手动操作**

1. 在 GitHub 创建新仓库：`xiaojunhong-website`
2. 运行以下命令：

```bash
git remote add origin https://github.com/你的用户名/xiaojunhong-website.git
git push -u origin main
```

### 第四步：连接 Cloudflare Pages

1. 访问 https://dash.cloudflare.com/
2. 进入 "Workers & Pages" → "Create application"
3. 选择 "Pages" → "Connect to Git"
4. 选择 `xiaojunhong-website` 仓库
5. 配置构建设置：
   - 构建命令：`hugo`
   - 构建输出目录：`public`
6. 点击 "Save and Deploy"

### 第五步：绑定域名

1. 在 Cloudflare Pages 项目中，添加自定义域名：`xiaojunhong.space`
2. 在 Namecheap 配置 DNS（详细步骤见《快速部署指南.md》）
3. 等待 DNS 生效（5-30 分钟）

## 🎨 自定义指南

### 修改网站标题
编辑 `config.toml`：
```toml
baseURL = "https://xiaojunhong.space/"
title = "你的网站标题"
```

### 修改颜色
编辑 `layouts/_default/baseof.html` 中的 CSS 变量：
```css
:root {
    --color-accent: #0066cc;  /* 主题色 */
}
```

### 添加导航链接
编辑 `config.toml` 中的 `[menu.main]` 部分。
